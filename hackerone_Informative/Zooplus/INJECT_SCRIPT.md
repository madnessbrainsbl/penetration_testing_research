# Инструкция: Выполнить тест в браузере

Поскольку я не могу напрямую выполнить JavaScript в консоли через MCP инструменты, выполните следующие шаги:

## Способ 1: Через консоль браузера (РЕКОМЕНДУЕТСЯ)

1. Откройте https://www.zooplus.de/checkout/cart
2. Убедитесь что залогинены под Account B
3. Откройте DevTools (F12) → Console
4. Скопируйте и выполните код из файла `test_inline.js`
5. Результат появится в консоли

## Способ 2: Через HTML файл

1. Откройте файл `create_data_url.html` в браузере
2. Убедитесь что залогинены под Account B на zooplus.de
3. Тест выполнится автоматически
4. Результат появится на странице и в консоли

## Ожидаемый результат:

- Если корзина изменилась → **Critical IDOR Write подтвержден!**
- Если не изменилась → нужно найти правильный формат payload
